# 🚀 Smart Task Manager

A modern task management web application built using **React** that helps users organize, track, and prioritize their daily tasks efficiently.

---

## 🔗 Live Demo
👉 https://smart-task-manager-orcin.vercel.app/

---

## 📌 Features

- ✅ Add, edit, and delete tasks
- 🎯 Mark tasks as completed
- 🔍 Filter tasks (All / Pending / Completed)
- ⚡ Priority levels (High / Medium / Low)
- 💾 Persistent storage using LocalStorage
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React (Functional Components + Hooks)
- **State Management:** useState, useEffect
- **Styling:** CSS
- **Deployment:** Vercel
- **Version Control:** Git & GitHub

---

## 🧠 Key Learnings

- Understanding component-based architecture in React
- Managing state and props effectively
- Implementing conditional rendering
- Handling user input and events
- Working with LocalStorage for data persistence
- Debugging real-world errors (deployment, Git, etc.)
- Deploying a production-ready application

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Codes-work/smart-task-manager.git
cd smart-task-manager
```
### 2. Install dependencies

npm install

### 3. Run the app

npm start
